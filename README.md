# مساعد السعرات والتقدم الصحي

تطبيق ويب عربي بسيط يعمل محلياً بدون تثبيت أي حزم إضافية.

## الميزات

- إدخال اسم الطعام بالعربية من قائمة جاهزة أو بكتابة حرة.
- إدخال العدد أو الوزن بالغرام.
- إدخال العمر والطول والوزن والجنس.
- حساب السعرات اليومية.
- حساب `BMI` و`BMR`.
- حفظ سجل التقدم محلياً داخل المتصفح.
- تحليل مبسط ونصائح غذائية موجهة لفقدان الوزن.

## التشغيل

افتح ملف `index.html` مباشرة في المتصفح.

## ملاحظات

- البيانات تُحفظ محلياً داخل المتصفح باستخدام `localStorage`.
- التحليل تقريبي ومخصص للاستخدام الشخصي، وليس بديلاً عن الطبيب أو أخصائي التغذية.
